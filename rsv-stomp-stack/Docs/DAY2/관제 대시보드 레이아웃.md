🛠️ DAY 2 심화: 실시간 그래프 적용 코드
현재 수신 중인 value를 실시간 꺾은선 그래프로 시각화하는 코드를 index.html에 추가해 보세요. (CDN으로 Chart.js를 활용합니다.)

HTML

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<div style="width: 80%; margin: auto;">
    <canvas id="realtimeChart"></canvas>
</div>

<script>
    const ctx = document.getElementById('realtimeChart').getContext('2d');
    const chart = new Chart(ctx, {
        type: 'line',
        data: {
            labels: [], // 시간축
            datasets: [{
                label: '로봇 이동 수치 (Value)',
                data: [],
                borderColor: '#007bff',
                tension: 0.3,
                fill: false
            }]
        },
        options: {
            scales: { y: { beginAtZero: true, max: 100 } },
            animation: false // 실시간성을 위해 애니메이션 끄기
        }
    });

    // STOMP 구독부 수정
    stompClient.subscribe('/topic/status', (response) => {
        const data = JSON.parse(response.body);
        
        // 데이터 추가 (최대 20개 유지)
        if (chart.data.labels.length > 20) {
            chart.data.labels.shift();
            chart.data.datasets[0].data.shift();
        }
        
        chart.data.labels.push(new Date().toLocaleTimeString());
        chart.data.datasets[0].data.push(data.value);
        chart.update(); // 차트 갱신
    });
</script>
📝 실무 필기: ROSbridge Suite와의 연결 고리
나중에 실제 로봇(ROS2)과 연결할 때는 다음과 같은 관례를 따릅니다.

데이터 소스: RobotSimulationScheduler 대신 실제 rosbridge_suite 노드가 띄워진 WebSocket 서버에서 데이터를 가져옵니다.

데이터 변환: Spring Boot의 **RobotService**에서 ROS2 메시지 형식을 지금의 RobotCommandDTO 형식으로 예쁘게 가공합니다.

무료 사용 확인: rosbridge_suite는 BSD 라이선스로, 회사 프로젝트에 포함하여 상업적으로 배포해도 비용이 전혀 발생하지 않는 완전 무료 오픈소스입니다.
